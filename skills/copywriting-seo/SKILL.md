# ✍️ Copywriting & SEO

Esta habilidade é a "Voz" do agente, responsável por transformar o contexto visual e estratégico em textos de alta conversão, legendas engajadoras e hashtags de performance.

## 🛠️ Tecnologias
- **IA Text Models**: Claude / Gemini para geração de copy.
- **CopyService.js**: Lógica de extração de contexto e formatação.

## 📁 Componentes
- `skills/copywriting-seo/scripts/CopyService.js`
- `skills/copywriting-seo/scripts/testar-copy.mjs`

## 🎯 Objetivos
- **Ganchos Magnéticos**: Criar frases iniciais que prendam a atenção no scroll.
- **Desenvolvimento Estratégico**: Manter a coerência com as imagens do carrossel.
- **CTAs Claras**: Chamadas para ação específicas para o nicho de Tráfego Pago.
- **Hashtags de Performance**: Seleção automática de tags relevantes.

## 📝 Padrões
- Uso de emojis estratégicos para facilitar a leitura.
- Espaçamentos corretos para evitar blocos de texto cansativos.
- Tom de voz autoritário, mas acessível (Especialista em Meta Ads).
